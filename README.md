# Calabash Android

Comandos básicos git:

```ruby
git pull - Traz a branch/conteúdo de algum lugar para outro lugar:
git pull
git pull origin master - Traz conteúdo da branch master (origin) para branch local

git clone - Clona o repositório:
git clone https://github.com/thiagomarquessp/calabash_aula_1.git

git checkout - b "NOME-Branch" cria uma nova branch:
git checkout -b "TP-Aula-1"

git status verifica o momento atual da sua branch:
git status

git add + arquivo adiciona arquivo a memória (antes de subir)
git add /workspace/calabash/calabash_aula_1/arquivo.rb

git commit -m "mensagem" faz commit (Coloque a mensagem)
git commit -m "Esse é um exemplo dos comandos basicos do git."

git push empurra para o repositório
git push 
```
